## üìù How It Works

The app uses a specific grading bracket to determine your success:

| Total Score | Grade | GP |
| :--- | :--- | :--- |
| 80+ | A+ | 4.00 |
| 75-79 | A | 3.75 |
| 70-74 | A- | 3.50 |
| 65-69 | B+ | 3.25 |
| 60-64 | B | 3.00 |
| 55-59 | B- | 2.75 |
| 50-54 | C+ | 2.50 |
| 45-49 | C | 2.25 |
| 40-44 | D | 2.00 |
| < 40 | F | 0.00 |

*Note: Calculations include a `0.001` buffer to ensure fair rounding at the bracket edges.*
